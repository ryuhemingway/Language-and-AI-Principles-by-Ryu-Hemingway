# Daily Brief

The original `brief` command generates a daily HTML study and intelligence
briefing.

```bash
brief
brief -p claude
brief -p openai
brief -p deepseek
brief -p local
brief --no-open
```

## Sections

- Anki analytics
- Calendar and deadline review
- Web intelligence
- Study optimizer
- Synthesis
- Fact check
- HTML rendering

## Optional Integrations

- AnkiConnect
- Apple Calendar on macOS
- `deadlines.json`
- `skills.json`
- `profile.md`
- Cloud or local AI provider

Generated HTML files are saved to `output/` and ignored by Git.
