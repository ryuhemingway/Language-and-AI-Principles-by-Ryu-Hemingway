# Transformers and Deep Learning Basics

This module teaches the minimum deep learning foundation needed to reason about
modern LLMs.

## Principles Included

- Neural network basics
- Gradient descent
- Attention
- Transformers
- Pretraining
- Fine-tuning
- Alignment
- Prompting vs RAG vs fine-tuning

## Why It Matters

Software engineers do not need to train frontier models to build AI products,
but understanding the basics helps with model choice, limitations, and system
design.

## Build Practice

Compare five scenarios and decide whether prompting, RAG, or fine-tuning is the
right first approach.
