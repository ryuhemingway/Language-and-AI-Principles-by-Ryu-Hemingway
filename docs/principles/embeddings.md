# Embeddings and Vector Databases

This module teaches semantic search foundations.

## Principles Included

- Embeddings as vector representations
- Similarity search
- Vector indexes
- Metadata filters
- Recall and precision
- Hybrid search
- Reranking

## Why It Matters

Embeddings power RAG, recommendation systems, deduplication, clustering, and
semantic search. AI engineers need to evaluate retrieval quality separately from
generation quality.

## Build Practice

Create a tiny vector record schema and design retrieval tests for 10 queries.
