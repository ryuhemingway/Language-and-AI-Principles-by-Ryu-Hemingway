# AI Deployment and MLOps

This module teaches how AI systems are shipped and maintained.

## Principles Included

- Serving
- Timeouts
- Retries
- Rate limits
- Fallbacks
- Experiment tracking
- Prompt/model/index versioning
- Monitoring
- Drift
- Human review
- Staged rollouts

## Why It Matters

AI systems fail in production in ways that ordinary software does not. Teams
need observability, rollback paths, and evaluation before and after release.

## Build Practice

Design a production dashboard for a RAG assistant with latency, cost, retrieval
quality, error rate, and user feedback metrics.
