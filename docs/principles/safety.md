# AI Safety, Security, and Privacy

This module teaches practical safety and security for AI applications.

## Principles Included

- Prompt injection
- Data exfiltration
- Unsafe tool use
- Least privilege
- Human approval
- Privacy
- Data governance
- Access control
- Bias
- Reliability
- Refusal and escalation rules

## Why It Matters

AI systems often connect language models to tools, private data, and external
actions. Engineers need threat models and guardrails before shipping.

## Build Practice

Write a threat model for a RAG assistant that can read private documents and
send emails.
