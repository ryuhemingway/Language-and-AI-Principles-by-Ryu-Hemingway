# Machine Learning Fundamentals

This module teaches the ML basics engineers need before building or evaluating
AI systems.

## Principles Included

- Supervised learning
- Labels and targets
- Loss
- Metrics
- Baselines
- Overfitting
- Generalization
- Data leakage

## Why It Matters

LLM apps still depend on ML thinking: datasets, metrics, evaluation, and
generalization. ML engineers need these foundations for every model lifecycle.

## Build Practice

Design a small classification dataset with train/validation splits, labels, and
metrics.
