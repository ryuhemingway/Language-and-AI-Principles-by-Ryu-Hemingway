# Local LLM Fundamentals

This module teaches how local models work and where they fit.

## Principles Included

- Local model tradeoffs
- Model size
- Quantization
- Context length
- Local servers
- OpenAI-compatible endpoints
- Local prompting constraints
- Local RAG and eval workflows

## Why It Matters

Local models are useful for privacy, offline work, lower marginal cost, and
experimentation. They are usually weaker than cloud models, so evaluation
matters.

## Build Practice

Run LM Studio locally, select a model in `Learn`, and ask the tutor a question
without using a cloud provider.
