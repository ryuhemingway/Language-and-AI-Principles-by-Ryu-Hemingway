# Agentic Workflows

Agentic workflows let a model plan, use tools, inspect results, and decide next
steps.

## Principles Included

- Agentic loops
- Tools
- State
- Memory
- Planning
- Routers
- Supervisors
- Guardrails
- Observability
- Max-step and cost limits

## Why It Matters

Agentic systems are useful for multi-step tasks, but they are harder to debug
and riskier than single model calls. Engineers need clear control flow and
boundaries.

## Build Practice

Build a small research agent with search, open-source, and summarize tools plus
a five-step limit.
