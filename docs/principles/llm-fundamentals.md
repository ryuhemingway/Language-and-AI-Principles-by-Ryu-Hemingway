# LLM and Model API Fundamentals

This module teaches what LLMs are useful for, how model APIs behave, and how to
choose models for engineering work.

## Principles Included

- LLM capabilities and limits
- Tokens, context windows, latency, and cost
- Structured output
- Tool calling
- Model selection
- API integration patterns

## Why It Matters

Software engineers need to know when to use a model and when normal code is the
better tool. AI engineers need to understand cost, latency, reliability, and
model fit before putting AI into products.

## Build Practice

Build a small CLI that sends a task to a model and returns structured JSON.
Validate the JSON before using it.
