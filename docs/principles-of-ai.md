# Principles of AI Track

The Principles of AI track is for software engineers, AI engineers, and ML
engineers who need to build practical AI systems.

```bash
Learn --track ai_principles
```

## Modules

| Module | Command |
| --- | --- |
| LLM and model API fundamentals | `Learn --track ai_principles --module llm_fundamentals` |
| Prompt and context engineering | `Learn --track ai_principles --module prompting` |
| Embeddings and vector databases | `Learn --track ai_principles --module embeddings` |
| RAG fundamentals | `Learn --track ai_principles --module rag` |
| Harnesses and evaluation | `Learn --track ai_principles --module harnesses` |
| Agentic workflows | `Learn --track ai_principles --module agents` |
| Local LLM fundamentals | `Learn --track ai_principles --module locallm` |
| Machine learning fundamentals | `Learn --track ai_principles --module ml_basics` |
| AI data engineering | `Learn --track ai_principles --module data_engineering` |
| Transformers and deep learning basics | `Learn --track ai_principles --module transformers` |
| AI deployment and MLOps | `Learn --track ai_principles --module mlops` |
| AI safety, security, and privacy | `Learn --track ai_principles --module safety` |

## Design Goal

The track is not a theory-only survey. Each module teaches:

- What the concept is
- Why it matters
- How it works in production systems
- How to build a small version yourself
- How to test or evaluate it
