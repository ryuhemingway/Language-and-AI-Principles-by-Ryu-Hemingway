# Programming Track

The programming track teaches Python, C, and Java from scratch.

## Languages

```bash
Learn --track programming --language python
Learn --track programming --language c
Learn --track programming --language java
```

## Included Concepts

- Variables and output
- Types and conversions
- Input
- Conditionals
- Loops
- Functions and methods
- Arrays/lists
- Strings
- Hash maps/dictionaries
- Stacks
- Two pointers
- Sliding window
- Binary search
- Classes and objects
- Linked lists
- Recursion
- Trees
- Graphs
- Dynamic programming
- Heaps
- Matrices
- Design problems

## Lesson Flow

Each lesson includes:

1. Concept explanation
2. Language-specific syntax note
3. Code example
4. Practice target
5. Quiz
6. Optional AI tutor help
7. LeetCode unlocks when prerequisites are complete

## Why These Topics

These topics cover the foundation needed for:

- Intro programming courses
- Data structures and algorithms
- Technical interviews
- LeetCode easy/medium practice
- Building small CLI apps and scripts
